# Sokha Care

Enterprise healthcare frontend foundation built with React, Vite, TypeScript, Tailwind CSS, React Router DOM, React Query, Axios, Zustand, React Hook Form, and Zod.

## What is included

- Clean Vite + React + TypeScript setup
- Tailwind CSS theme foundation
- Absolute imports with `@/`
- Environment variable support
- ESLint and Prettier configuration
- React Router route architecture
- React Query provider setup
- Axios API client with interceptors
- Zustand auth store
- Reusable layouts and UI components
- Protected and public route guards
- Lean modular frontend structure without dead scaffolding

## Main scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run format`
- `npm run typecheck`

## Quick start

1. Install dependencies with `npm install`.
2. Copy `.env.example` to `.env` and adjust API values as needed.
3. Start the app with `npm run dev`.

## Folder structure

The app foundation lives under `src/` with modular areas for:

- `app/`
- `components/`
- `config/`
- `constants/`
- `layouts/`
- `pages/`
- `routes/`
- `services/`
- `store/`
- `styles/`
- `types/`
- `utils/`
Sokha Care is an AI-powered, multilingual digital healthcare ecosystem designed to improve healthcare accessibility through telemedicine, smart consultations, appointment management, emergency healthcare support, pharmacy integration, and scalable healthcare infrastructure for urban, semi-urban, and rural communities.
